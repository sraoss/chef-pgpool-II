chef-pgpool-II
==============
This cookbook installs pgpool-II and pgpoolAdmin and manages their configurations.
It uses [pgpool-II Tutorial](http://www.pgpool.net/pgpool-web/contrib_docs/watchdog_master_slave_3.3/en.html). as a reference.

Version
-------
0.1 alpha1

Requirements
------------
* CentOS / RHEL
* Chef: 11
* ruby >= 1.9
